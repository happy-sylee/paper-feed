# ===================================================================
# 카테고리별 주제 키워드
# ===================================================================
# 형식:
#   [축:카테고리]  <- 대괄호로 카테고리 헤더 선언
#   키워드          <- 그 아래 한 줄에 하나씩
#
# 규칙:
#   - 축은 SYSTEM(배터리 시스템) 또는 COMPONENT(구성요소)
#   - 논문 제목+초록에 키워드가 있으면 그 카테고리로 태깅됨
#   - 한 논문이 여러 카테고리에 동시 태깅 가능 (예: LIB + Cathode)
#   - 대소문자 구분 안 함
#   - '#'으로 시작하는 줄, 빈 줄은 무시
#   - 어느 SYSTEM에도 안 걸리면 'others'(시스템), COMPONENT도 마찬가지
#
# ★ 필터링 기준: 아래 키워드 중 하나라도 매칭되면 그 논문은
#   "battery 관련"으로 간주되어 피드에 표시됨.
#   (즉, 이 파일이 곧 수집 필터 역할도 함)
#
# ⚠️ 약어 주의: 너무 짧고 흔한 약어(Al, C, Na 단독 등)는 오탐 위험.
#   가능하면 구체적 표현(sodium-ion 등)을 함께/대신 사용.
# ===================================================================
