# 형식:  그룹 | 저널표시이름 | RSS주소
#   - 한 줄에 하나씩, '|' 로 3칸 구분
#   - '그룹'은 왼쪽 사이드바 탭으로 묶이는 단위 (예: Nature, Wiley, ACS)
#   - 같은 그룹 이름끼리 자동으로 묶임
#   - '#'으로 시작하는 줄, 빈 줄은 무시
#   - 저널 빼려면 줄 앞에 '#' 붙이거나 삭제
# ===================================================================
